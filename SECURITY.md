# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in any INORSA repository, please report it responsibly.

**Do not open a public issue.**

Email: security@inorsa.com

We will acknowledge your report within 48 hours and provide a timeline for a fix.

## Scope

This policy applies to all repositories under the [INORSA](https://github.com/INORSA) GitHub organization. If a specific repository has its own `SECURITY.md`, follow that instead.

## Automated Security Monitoring

INORSA runs automated security audits via [org-management](https://github.com/INORSA/org-management) that check Dependabot alerts, secret scanning, push protection, and CODEOWNERS compliance across all repositories. These audits complement — but do not replace — responsible vulnerability disclosure through the process above.
